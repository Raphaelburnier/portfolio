# Portfolio

Personal portfolio site for Raphael Burnier, data scientist and full-stack developer based in Lausanne.

Single-page site built with HTML and Tailwind CSS (via CDN). No build step: `index.html` is self-contained and can be opened directly or served statically.

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.

## Sections

About, Education (degrees and certifications), Skills, Projects, Experience, Contact.
